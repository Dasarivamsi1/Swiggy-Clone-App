# Swiggy-Clone-App
Swiggy-clone-App Devops Project using Jenkins, Docker , AWS, AWS EKS, Git, Github, Trivy and Sonarqube

Code Developed and taken help from Chatgpt ---> Terraform ( Used to create EC2 instance, VPC ) ---> From EC2 instance Public ip ,Loginned to Mobaxterm for remote computing ---> Github ( Pushed the code from remote repo ) -----> Jenkins ( Built a CI/CD pipeline and automated the build that code changes done on github ) ---> SonarQube ( Static code Analysis ) ---> Trivy ( security scanning to detect Vulnarabilities from Artifacts ) ---> Docker image ( Built Docker image and  pushed to dockerhub ) ---> Deployed on AWS EKS ( Configured Kubernetes in AWS and Used yaml for deployment ).
