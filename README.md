# Swiggy-Clone-App
Swiggy-clone-App Devops Project using Jenkins, Docker , AWS, AWS EKS, Git, Github, Trivy and Sonarqube

Code Developed and taken help from Chatgpt ---> Github ( Pushed the code from remote repo ) -----> Jenkins ( Built a CI/CD pipeline and automated the build that code changes done on github ) ---> SonarQube ( Static code Analysis ) ---> Trivy ( security scanning to detect Vulnarabilities from Artifacts ) ---> Docker image ( pushed to docker hub ) ---> Deployed on AWS EKS ( Configured Kubernetes in AWS and deployed the application ).
